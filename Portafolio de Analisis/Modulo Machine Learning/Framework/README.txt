Proyecto: Framework de Regresión Logística - Week02 Challenge2

Este repositorio contiene el proyecto titulado "Framework de Regresión Logística", desarrollado como parte del Week02 Challenge2. El archivo principal del proyecto es el notebook "Framework_AdrianPineda_RegresionLogistica_Framework_Week02_Challenge2.ipynb".

Archivos Clave:

1. Framework_AdrianPineda_RegresionLogistica_Framework_Week02_Challenge2.ipynb
   Este es el archivo principal del proyecto. Contiene todo el desarrollo del modelo de regresión logística, incluyendo:
   - Exploración de datos: Análisis exploratorio inicial.
   - Preprocesamiento de datos: Preparación de los datos para el modelo.
   - Entrenamiento del modelo: Construcción y ajuste de la regresión logística.
   - Evaluación del modelo: Métricas de desempeño y comparación de resultados.

2. Framework_AdrianPineda_RegresionLogistica_Framework_Week02_Challenge2.html
   Este archivo es la versión convertida a HTML del notebook. Puedes abrirlo en cualquier navegador web para revisar el contenido del notebook sin necesidad de un entorno Jupyter.

3. Challenge02_02.png
   Esta imagen hace alusión al reto Week02 Challenge2 y puede ser utilizada como referencia visual del proyecto.

4. README.txt
   Este archivo que estás leyendo. Proporciona una visión general del proyecto, así como una guía de los archivos clave que se deben revisar.
