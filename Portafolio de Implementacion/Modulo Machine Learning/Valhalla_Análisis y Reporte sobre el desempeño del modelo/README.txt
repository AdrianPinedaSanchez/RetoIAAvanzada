Proyecto: Análisis de Datos - Valhalla
Autor: Adrian Pineda

Descripción
Este repositorio contiene el análisis de datos realizado en el notebook titulado Valhalla_AdrianPineda.ipynb. El análisis incluye la preparación de datos, modelado utilizando regresión lineal, evaluación de modelos y visualización de resultados.

Archivos a Revisar:

*Valhalla_AdrianPineda.ipynb: Este es el archivo principal que contiene todo el desarrollo del proyecto. Incluye código comentado, análisis, gráficos y conclusiones.
*Valhalla_AdrianPineda.html: Versión en formato HTML del notebook para una visualización rápida en el navegador.
*Valhalla_AdrianPineda.pdf: Versión en formato PDF del notebook para una revisión en formato estático.
*Valhalla23.csv: Base de datos utilizada en el proyecto, que contiene los datos que fueron analizados y modelados.
